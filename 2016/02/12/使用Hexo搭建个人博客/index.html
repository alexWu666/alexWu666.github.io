使用Hexo搭建个人博客

发表于 2016-02-12   |   阅读次数

实验室的学长们在寒假布置我们通过博客来记述自己的学习过程。由于服务器的价格太贵，所以我采用腾讯云学生免费域名+Hexo+github来搭建自己的第一个博客。

###入门知识

##Hexo
Hexo是一个功能强大的静态网站生成系统，快速、简洁、高效。Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。可以一键部署到github，还有丰富的插件和主题,还支持热部署哦。执行如下命令进行安装。

##node.js
因为Hexo是基于node.js的，所需要到node.js的官网下载安装。（安装默认步骤就好了。）

##git
Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。提供一个”用户名“.github.io的个人主页，共大家使用。

###实战操作

##网络上关于Hexo搭建的教程很多（例如：HEXO+Github,搭建属于自己的博客）。我就是按照这个搭建而成的。现在我总结一下自己的错误地方共大家参考。1.需要新建一个文件夹然后再右击它，选择GIT GUI HERE.2.上传的时候第一次填入的是usename,第二次才是密码。其实这应该是搭建博客最简单的方式。

###总结

##通过这次搭建博客，我完成了自己第一篇博客，虽然内容很空洞，但是因为这个真的很简单。
